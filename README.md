# Weather App using React
---
Developed a Weather React.JS App using the Open Weather Map API
The background is set to change to the background image of the application to a image representing Cold weather if the temp of the location entered is below 70Â°F.
Once the user enters the desired location, the application displays 
- Current date 
- City name
- Temperature
- Weather Icon
- Weather Description
- What the temperature it feels like
- Humidity 
- Wind Speed
